🔐 萬用 Cloudflare Worker Proxy 範本專區 ▼

所有需要隱藏 API Key 或解決 CORS 的服務（Claude、OpenAI、Notion、Twilio、SendGrid）均可共用以下這套 Worker 模板。將 Key 設為 Wrangler Secret，前端只需呼叫你的 Worker URL。

// === 萬用 API Proxy Worker（wrangler deploy）=== // 支援：Claude / OpenAI / Notion / SendGrid / Twilio const ROUTES = { '/claude': { target: 'https://api.anthropic.com/v1/messages', secretKey: ANTHROPIC_KEY, extraHeaders: { 'anthropic-version': '2023-06-01' } }, '/openai': { target: 'https://api.openai.com/v1/chat/completions', authPrefix: 'Bearer', secretKey: OPENAI_KEY }, '/notion': { target: 'https://api.notion.com/v1', authPrefix: 'Bearer', secretKey: NOTION_KEY, extraHeaders: { 'Notion-Version': '2022-06-28' } }, '/sendgrid': { target: 'https://api.sendgrid.com/v3/mail/send', authPrefix: 'Bearer', secretKey: SENDGRID_KEY }, }; export default { async fetch(req, env) { const url = new URL(req.url); const route = Object.entries(ROUTES).find(([p]) => url.pathname.startsWith(p)); if (!route) return new Response('Not found', { status: 404 }); const [path, cfg] = route; const secret = env[cfg.secretKey?.name ?? cfg.secretKey]; const authVal = cfg.authPrefix ? `${cfg.authPrefix} ${secret}` : secret; const headers = { 'Content-Type': 'application/json', ...(cfg.extraHeaders ?? {}) }; if (cfg.authPrefix) headers['Authorization'] = authVal; else headers['x-api-key'] = secret; const res = await fetch(cfg.target + url.pathname.slice(path.length), { method: req.method, headers, body: req.body }); return new Response(res.body, { status: res.status, headers: { 'Access-Control-Allow-Origin': '*', 'Content-Type': 'application/json' } }); } };

以下為各 API 對應的 Secret 名稱清單，執行 wrangler secret put <NAME> 設定：
